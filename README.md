# Portfolio Website

This is a personal portfolio website showcasing projects, skills, and contact information.

## Features

- **Responsive Design:** Adapts to different screen sizes for a seamless experience on desktop and mobile devices.
- **Light/Dark Theme:** Switch between light and dark modes for user comfort.
- **Interactive UI:** Smooth animations and transitions for a modern feel.
- **Project Showcase:** A grid layout to display featured projects with details and links.
- **Skills Section:** Highlights technical expertise and tools.
- **Contact Information:** Easy access to social media profiles and email.

## Technologies Used

- **HTML5:** For the structure and content of the website.
- **CSS3:** For styling and layout, including Flexbox and Grid.
- **JavaScript (ES6+):** For interactive features like the theme switcher and smooth scrolling.
- **Font Awesome:** For icons used throughout the website.
- **Google Fonts:** For typography.

## How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd your-repo-name
   ```
3. **Open `index.html` in your web browser.**

You can customize the content by editing the `index.html`, `style.css`, and `script.js` files.

## Customization

- **`index.html`:** Modify the text, project details, and links to reflect your own information.
- **`style.css`:** Change colors, fonts, and layout to match your personal brand.
- **`script.js`:** Add or modify JavaScript functionality as needed.
